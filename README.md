# 软件库-用户界面和管理员界面

#### 介绍

这是一个提供软件下载的网站导航模板。虽然样式简单，但仍支持前端上传软件。目前暂未开发站点信息前端更改功能，需要进入服务器后台修改源文件。

#### 使用方法

具体使用方法请参考 [AiQiji](https://aiqji.com)。
